Loads and merges training metadata with full essay texts.

Reads train.csv and corresponding essay .txt files, attaches full essay text to each discourse row, and maps effectiveness labels to integers.

Dataset module for the Feedback Prize 2 task.

This module handles the conversion of a pandas DataFrame into a PyTorch-compatible dataset with tokenized input suitable for transformer-based models.

Classes:

Aggregates all discourse units in a single essay into one formatted string.

Concatenates discourse texts prefixed by a custom [FP2] token, and stores associated metadata and labels for the essay.

Parameters:
