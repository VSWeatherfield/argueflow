argueflow package

This module provides inference functions for the ArgueFlow pipeline, including utilities to load models, run predictions, and organize outputs.

This module handles model training, validation, and fold-wise setup for the ArgueFlow pipeline.

Aggregates all discourse units in a single essay into one formatted string.
